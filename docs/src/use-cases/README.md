# Use cases

These are concrete starting points — real configurations you can copy, adjust, and run.

Each example shows you a `coulisse.yaml` and what it gives you. None of them require anything beyond a single `coulisse start`.

- [Personal assistant with memory](./personal-assistant.md) — remembers facts about you between sessions
- [Multi-model app](./multi-model.md) — fast triage, slow reasoning, same endpoint
- [Slack / webhook bot](./webhook-bot.md) — any HTTP source triggers an agent
- [Daily briefing (cron)](./daily-briefing.md) — an agent wakes up every morning without any user action
- [Orchestrator + specialists](./orchestrator-specialists.md) — one agent dispatches to focused sub-agents
